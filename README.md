# PRIME XI - Mobile-First Optimization 🏆

**PRIME XI** es la plataforma de análisis avanzado y optimización para la Fantasy Premier League, diseñada para mánagers que exigen la victoria desde la palma de su mano.

---

## 🎨 El Objetivo: "Mobile-First Midnight Precision"
Nuestro norte es el diseño premium adaptado al ritmo de vida actual. No creamos una web que "funcione" en móvil; creamos una experiencia nativa web diseñada desde el píxel 1 para smartphones.

### Pilares de Diseño:
- **Midnight Navy Aesthetic:** Dark mode sofisticado con efectos de glassmorphism controlados.
- **Thumb-Driven UI:** Navegación y acciones críticas situadas en la zona de fácil alcance del pulgar.
- **Vertical Tactical Pitch:** Una visualización del campo de juego optimizada para el scroll vertical.
- **Micro-interacciones Táctiles:** Feedback hápitco visual mediante cambios de estado reactivos.

---

## 🛠️ Stack Tecnológico (Optimizado para Performance)
- **Framework:** [Next.js (Pages Router)](https://nextjs.org/) - Renderizado rápido y rutas simples.
- **Base de Datos:** [Supabase](https://supabase.com/) - Escalabilidad y tiempo real.
- **Estilos:** [Tailwind CSS](https://tailwindcss.com/) - Diseño responsivo con enfoque Mobile-First (`min-width: 0px`).
- **Animaciones:** [Framer Motion](https://www.framer.com/motion/) - Transiciones fluidas de página y componentes.
